CSS Cheatsheet

A handy reference for important CSS tags


CSS Properties


    Color and Background

  • color - Определя цвета на текста в елемент
  • background-color - Задава фоновия цвят на елемент
  • background-image - Поставя изображение като фон на елемент
  • background-size - Задава размера на фоновото изображение
  • opacity - Регулира прозрачността на елемент

    Text and Font

  • font-family - Определя шрифта на текста в елемент, обикновено с набор от имена на шрифтове за резервни варианти.
  • font-size - Задава размера на шрифта в елемент
  • font-weight - Контролира дебелината или тежестта на шрифта
  • line-height - Определя разстоянието между редовете текст, често за подобряване на четимостта.
  • text-align - Подравнява текста хоризонтално
  • text-decoration - Добавя украси като underline (подчертаване), overline или line-through (зачеркване) на текста.
  • text-transform -Контролира преобразуването на текста, като например прави всички букви главни (uppercase), малки (lowercase) или само първата главна (capitalize).
  • letter-spacing - Определя разстоянието между отделните букви.
  • word-spacing - Определя разстоянието между думите в текста.

    Box Model

  • margin - Определя цвета на текста в елемент
  • padding - Задава фоновия цвят на елемент
  • border - Поставя изображение като фон на елемент
  • border-radius - Задава размера на фоновото изображение
  • box-shadow - Регулира прозрачността на елемент
  • width - Определя цвета на текста в елемент
  • height - Задава фоновия цвят на елемент
  • max-width - Поставя изображение като фон на елемент
  • max-height - Задава размера на фоновото изображение
  • min-width - Регулира прозрачността на елемент
  • min-height - Регулира прозрачността на елемент

    Positioning and Layout

  • display - Определя как се показва елементът в документа
  • position - Задава метода на позициониране на елемента
  • top/right/bottom/left - Определят отместването на елемент, ако е позициониран различно от static
  • float - Използва се за поставяне на елемент вляво или вдясно, за да се "обтека" от другото съдържание
  • clear - Определя как други елементи трябва да се позиционират спрямо елемент с float
  • flex - Активира режим на гъвкаво оформление (flexbox) за родителски елемент
  • grid - Активира мрежово оформление (grid) за елемент и задава специфични редове и колони

    Visibility and Overflow

  • visibility - Контролира видимостта на елемент
  • overflow - Определя какво се случва, когато съдържанието надхвърли размера на контейнера
  • overflow-x - Задава преливане по хоризонтала
  • overflow-y - Задава преливане по вертикала

    Transitions and Animations

  • transition - Задава ефект на преход между начално и крайно състояние на свойство
  • transition-duration - Определя продължителността на прехода
  • animation - Задава анимация за елемент
  • animation-duration - Определя продължителността на анимацията

CSS Selectors

    Universal Selector

  • * - Избира всички елементи на страницата

    Type Selector

  • div/p/h1 - Избира всички елементи от даден тип

    ID Selector

  • #idname - Избира един елемент със съответния идентификатор (уникален за документа)

    Class Selector

  • .classname - Избира всички елементи с конкретен клас

    Pseudo-classes

  • hover - Приложен към елемент, когато потребителят премине с курсора върху него
  • active - Приложен към елемент, когато потребителят го кликне
  • focus - Приложен към елемент, когато той е фокусиран (често за форми и бутони
